## Workflow

1. **Spec schreiben** — Klartext-Datei mit Sektionen, Texten, Formularfeldern
2. **CORPORATE-DESIGN.md** referenzieren — Farben, Logo, Fonts, Layout
3. **HTML generieren** — aus Spec + CD wird eine standalone Landing Page gebaut
4. **Neue Version** — Änderungen → `v002/`, `v003/` etc.

## Corporate Design

Siehe [CORPORATE-DESIGN.md](CORPORATE-DESIGN.md) für:
- Logo (inline SVG, graue Sprechblasen)
- Farbpalette (Primary Lila #5755d9, Grün nur WhatsApp)
- System-Fonts (kein Google Fonts)
- Layout-Stil und Seitenstruktur
